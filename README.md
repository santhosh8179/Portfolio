# Santhosh Panakanti — Portfolio

React + Vite portfolio. Ready to deploy on Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this folder to a Git repo (e.g. GitHub).
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import the repo.
3. Vercel will detect Vite; keep **Build Command**: `npm run build`, **Output Directory**: `dist`.
4. Click **Deploy**.

Or with Vercel CLI from the `portfolio` folder:

```bash
npm i -g vercel
vercel
```

Follow prompts and deploy.
